# Smart File Organizer Pro

## Overview

Smart File Organizer Pro is a Python automation tool that automatically organizes files into categorized folders based on their file extensions.

## Features

* Automatically detects file types
* Creates folders dynamically
* Moves files into appropriate categories
* Generates activity logs
* Displays organization summary

## Technologies Used

* Python
* OS Module
* Shutil Module
* Datetime Module

## Supported Categories

* Images
* Documents
* Videos
* Spreadsheets
* Archives

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:
   python main.py
4. Enter the folder path.
5. Files will be organized automatically.

## Output

* Organized folders based on file type
* Activity log file (organizer_log.txt)
* Summary of files organized
